### Реализация асинхронного краулера для Hacker News

Асинхронный краулер для сайта https://news.ycombinator.com/, реализованный с использованием asyncio, aiohttp и BeautifulSoup.

Программа регулярно обходит главную страницу Hacker News, отслеживает новые посты в топе и скачивает:

HTML страницы самой новости
HTML страницы по ссылкам, найденным в комментариях

Каждая новость сохраняется в отдельной директории на диске.

#### Установка и запуск 


```
poetry install
python main.py 
```
